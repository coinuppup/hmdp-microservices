# 接口时序图

## 1. 用户服务接口

### 1.1 发送验证码

```mermaid
sequenceDiagram
    participant Client as 客户端
    participant UserService as 用户服务
    participant Redis as Redis
    participant SMS as 短信服务

    Client->>UserService: POST /user/code (phone)
    UserService->>Redis: 检查验证码发送频率
    alt 发送频率过高
        Redis-->>UserService: 返回错误
        UserService-->>Client: 429 Too Many Requests
    else 发送频率正常
        UserService->>UserService: 生成验证码
        UserService->>Redis: 存储验证码 (key: code:{phone}, value: {code}, expire: 5min)
        UserService->>SMS: 发送短信
        SMS-->>UserService: 发送成功
        UserService-->>Client: 200 OK {"success": true, "message": "验证码发送成功"}
    end
```

### 1.2 用户登录

```mermaid
sequenceDiagram
    participant Client as 客户端
    participant UserService as 用户服务
    participant Redis as Redis
    participant MySQL as 数据库
    participant JWT as JWT服务

    Client->>UserService: POST /user/login (phone, code)
    UserService->>Redis: 获取验证码
    alt 验证码不存在或已过期
        Redis-->>UserService: 返回错误
        UserService-->>Client: 400 Bad Request {"error": "验证码错误"}
    else 验证码存在
        UserService->>Redis: 删除验证码
        UserService->>MySQL: 查询用户
        alt 用户不存在
            MySQL-->>UserService: 返回空
            UserService->>MySQL: 创建新用户
            MySQL-->>UserService: 创建成功
        end
        UserService->>JWT: 生成token
        JWT-->>UserService: 返回token
        UserService-->>Client: 200 OK {"success": true, "data": {"token": "...", "user": {...}}}
    end
```

### 2.10 秒杀下单

```mermaid
sequenceDiagram
    participant Client as 客户端
    participant ShopService as 购物服务
    participant Redis as Redis
    participant Kafka as Kafka
    participant MySQL as 数据库

    Client->>ShopService: POST /voucher-order/seckill/:id (Authorization: Bearer {token})
    ShopService->>ShopService: 获取用户ID
    ShopService->>Redis: 检查是否已下单
    alt 已下单
        Redis-->>ShopService: 返回存在
        ShopService-->>Client: 400 Bad Request {"error": "禁止重复下单"}
    else 未下单
        ShopService->>Redis: 扣减库存
        alt 库存不足
            Redis-->>ShopService: 返回负数
            ShopService->>Redis: 恢复库存
            ShopService-->>Client: 400 Bad Request {"error": "库存不足"}
        else 库存充足
            ShopService->>Redis: 添加到已下单集合
            ShopService->>ShopService: 生成订单ID
            ShopService->>Kafka: 发送订单消息
            Kafka-->>ShopService: 发送成功
            ShopService-->>Client: 200 OK {"success": true, "data": {"id": 1}}
            Kafka->>ShopService: 消费订单消息
            ShopService->>MySQL: 扣减库存
            ShopService->>MySQL: 创建订单
        end
    end
```
